B) Directories & File Uses

Our Design uses only assets & lib folder. So, don't touch any other folder for ui changes.

    • assets(Folder) - Put all Static images & languages file in this folder
    • fonts(Folder) - Put all fonts file in this folder
    • languages(Folder) - Put all languages file in this folder
    • lib(Folder) - Main folder that contains all the dart code
      • localization(Folder) - Contain localization related file in this folder.
      • screens(Folder) - Contain All App Screens in this folder except main.dart.
      • theme(Folder) - Contain theme (Constant colors, fontsize, textstyle, etc) file in this folder.
      • widget(Folder) - Contain our own created widget file in this folder.
      • main.dart - Entry Point of Our App.
  • pubspec.yaml(File) - Specification file (All the package and local assets (Like Images, Fonts, etc.) we use in this application is specify in this file.).
